## Conky
How to install conky:

sudo apt-get install conky

Put conky.conf in 
~/.config/conky/conky.conf

Put conky.desktop in 
~/.config/autostart/conky.desktop

tutorial:
https://www.youtube.com/watch?v=QB8cjKpdVQY&t=523s
